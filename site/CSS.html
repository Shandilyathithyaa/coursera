Selectors

  • Element
  • Class(.name)
  • Id(#name)

Combining selectors

  • selector with class. e.g p.big
  • Child selector: Immediate child e.g. Selector1 > Selector2(Also for class)
  • Descendant: similar to child but at any level: Sel1 Sel2
  • Adjacent sibling: selector + selector
  • General sibling: selector ~ selector

Pseudo-Class Selectors

  • link
  • visited
  • hover
  • active
  • nth_child

Style placement

Style can be placed inline using style="" but should be done only for quick testing. External style sheets can be created and linked to the page by using <link rel="stylesheet" href="">

Box model

The model has Margin, border, padding, and then content.
The width styling works only for content box; But for it to work for the whole model, USE box-sizing: border-box
width=500=>5+10+470+10+5
NOTE: box-sizing is not inheritable and to apply for all elements, USE *{}
for two boxes:
  • side-side->cumulative margin
  • top-bottom->dominant margin
For overflowing content, USE overflow: auto to include automatic scrollbar; NOTE: Users hate double scrolling.
